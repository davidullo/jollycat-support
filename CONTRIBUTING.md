# Contributing

This repository is for **bug reports, feature requests, and support** only. Jollycat is a closed-source application â€” no source code is hosted here.

## How to Contribute

- **Report bugs** using the Bug Report template
- **Suggest features** using the Feature Request template
- **Report conversion issues** using the Conversion Issue template
- **Help other users** by commenting on open issues with solutions or workarounds

## Before Submitting

1. Search [existing issues](https://github.com/davidullo/jollycat-support/issues) to avoid duplicates
2. Use the appropriate issue template
3. Include as much detail as possible (version, macOS version, steps to reproduce)

Thank you for helping make Jollycat better!
