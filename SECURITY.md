# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Jollycat, please report it responsibly.

**Do not open a public issue for security vulnerabilities.**

Instead, email us at: **security@jollycat.app**

We will acknowledge your report within 48 hours and work with you to understand and address the issue.

## Scope

This policy covers the Jollycat desktop application. For issues related to third-party services (LemonSqueezy, GoFile.io), please contact those services directly.
